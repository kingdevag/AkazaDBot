Comandos

El prefijo del Bot es

!

El prefijo se utiliza antes de cada comando, Ejemplo: !help

game

Utilizalo para jugar 3 en raya o conocido en algunos paises como gato

lugar

"Utilizalo despues de !game para poner en casilla, Ejemplo: !game !lugar 3

name_edit

Cambia el nombre del etiquetado, ejemplo: !name_edit @akaza

secret

Manda un mensaje sin nombre del autor

Hola

Saludas al Bot

Proporciona informacion sobre el Servidor

subs

Proporciona el numero de subscriptores, Ejemplo: !subs HolaSoyGerman

youtube

Funciona como un buscador de youtube

sumar

Suma 2 numeros diferentes

help

Sirve para pedir ayuda sobre el bot Ya sea sobre comandos, informacion o funciones

multiplicar

multiplica 2 numeros diferentes

restar

resta 2 numeros diferentes

dividir

divide 2 numeros diferentes

user_info

Proporciona informacion sobre un usuario

invite

Con el puedes invitar al Bot a otro servidor

configbienvenida

Configura el canal de texto introducido para las bienvenidas, Puedes mencionar el canal o poner su id para estableserlo como canal de bienvenidas
